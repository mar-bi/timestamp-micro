FCC: Timestamp Microservice

User Stories

  1. I can pass a string as parameter and it will check whether the string contains either a unix timestamp or a natural language date.
  2. If it does, it returns both the unix timestamp and the natural language representation of that date.
  3. If it does not, returns null for both properties.

Examples of Usage

  • https://stitch-zoo.glitch.me/October 31, 2017
  • https://stitch-zoo.glitch.me/Jan%31,%2018
  • https://stitch-zoo.glitch.me/1509408000000
  • https://stitch-zoo.glitch.me/hello

Example output

{ "unix": 1509408000000, "natural": "October 31, 2017" }
or
{ "unix": null, "natural": null }
